# CKD-CDM-CVD Multimorbidity Analysis 🏥

A machine learning–based thesis project that combines three major clinical datasets — **Chronic Kidney Disease (CKD)**, **Chronic Diabetes Mellitus (CDM)**, and **Cardiovascular Disease (CVD)** — to study comorbidity patterns, shared risk factors, and predictive modeling for multimorbidity.

---

## 📌 Project Overview

This research explores the intersection of three chronic conditions that commonly co-occur in patients. By merging and analyzing CKD, CDM, and CVD datasets, this project aims to:

- Identify shared clinical features and risk factors across diseases
- Build classification/prediction models for single and multi-disease outcomes
- Analyze comorbidity relationships using statistical and ML methods
